================================================================
  仓储物资出入库动态管理系统 - Windows 7 版本
================================================================

【系统要求】
- 操作系统: Windows 7 SP1 或更高版本
- 内存: 2GB 以上
- 硬盘: 500MB 可用空间

【部署方式】

方式一：脚本启动（需要安装 Python 和 Node.js）
1. 安装 Python 3.8: https://www.python.org/ftp/python/3.8.10/python-3.8.10.exe
2. 安装 Node.js 16.x: https://nodejs.org/dist/v16.20.2/node-v16.20.2-x86.msi
3. 双击运行 start_win7.bat
4. 按提示完成配置

方式二：打包为 EXE（无需安装任何环境）
1. 在开发机运行: python build_win7.py
2. 将生成的 "仓储系统_Win7_完整版.zip" 复制到客户电脑
3. 解压后双击 "启动仓储系统.exe"

【默认账号】
- 用户名: admin
- 密码: admin123

【访问地址】
- 系统界面: http://localhost:3003
- API文档: http://localhost:8003/docs

【数据备份】
- 数据库文件: data/warehouse.db
- 直接复制此文件即可备份

【技术支持】
详细说明请查看: Windows7部署说明.md
